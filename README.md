# homebrew-clicher

Homebrew tap for [Clicher](https://github.com/naoki-mrmt/Clicher) - macOS screenshot & annotation tool.

## Install

```bash
brew tap naoki-mrmt/clicher
brew install --cask clicher
```

## Update

```bash
brew upgrade --cask clicher
```

## Uninstall

```bash
brew uninstall --cask clicher
```
